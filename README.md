#### Aubrey Johnston
<br>

## Technologies Used
<ul>
    <li>Node.js</li>
    <li>Express</li>
    <li>Webpack</li>
    <li>IndexedDB</li>
</ul>    

<br>

## Description

This application is designed to be a place where users can write code (or anything) and have that text be saved within this text editor, even after the page has refreshed or closed.
